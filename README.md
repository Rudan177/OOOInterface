# OOOInterface

一个用于替换浏览器新标签页的 Chrome 扩展插件。

- **产品名称**: OOOInterface5.1
- **当前版本**: 5.1.2:24.3-RS42.3

- **主要开发者**: RUDAN

## 功能描述

本插件会在您打开浏览器新标签页时，显示一个自定义的首页界面，替代浏览器默认的新标签页。

## 安装方法

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目所在文件夹
